# ZedYor Project

This project contains database setup scripts and feature implementations for KMUTT 2-2.

## Project Structure

- `setup/`: SQL scripts for creating and populating tables.
- `src/`: Source code for new features (e.g., JavaScript, CSS).

## Git Workflow for Active Branches

To ensure your work appears in the **Active branches** list on GitHub:

1. **Create a new branch** for your feature:
   ```bash
   git checkout -b feat/your-feature-name
   ```
2. **Make your changes** in the `src/` directory.
3. **Commit and Push**:
   ```bash
   git add .
   git commit -m "Add your descriptive commit message"
   git push origin feat/your-feature-name
   ```
